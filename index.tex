\subsection{Suppress cross-reference
prefix}\label{suppress-cross-reference-prefix}

If write \texttt{see\ @sec-lorem-ipsum-this} it will produce ``see
Section~\ref{sec-lorem-ipsum-this}''.

To suppress cross-reference prefix prepend \texttt{-} before \texttt{@},
like so \texttt{{[}-@citation{]}}, will suppress default prefix,
e.g.~\texttt{{[}-@sec-lorem-ipsum-this{]}} will produce just
\texttt{2.1} (or whatever number this section happens to be) without
\texttt{Section} prefix:

\begin{Shaded}
\begin{Highlighting}[]
\NormalTok{See }\CommentTok{[}\OtherTok{{-}@sec{-}lorem{-}ipsum{-}this}\CommentTok{]}\NormalTok{.}
\end{Highlighting}
\end{Shaded}

to get

See \ref{sec-lorem-ipsum-this}.

\subsection{To have another CR prefix}\label{to-have-another-cr-prefix}

To have another prefix and choose a momentary one at will, such as
`Chapter' or `Appendix', you must put the reference to the CR in square
brackets, preceded by the prefix you have chosen (in this example it is
`Chapter'):

\begin{Shaded}
\begin{Highlighting}[]
\NormalTok{See }\CommentTok{[}\OtherTok{Chapter @sec{-}lorem{-}ipsum{-}this}\CommentTok{]}\NormalTok{ for more details on visualizing model diagnostics.}
\end{Highlighting}
\end{Shaded}

to get

See Chapter~\ref{sec-lorem-ipsum-this} for more details on visualizing
model diagnostics.
